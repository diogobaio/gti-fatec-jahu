# Sistema Petshop

Este é um sistema simples de gerenciamento para Petshop desenvolvido em PHP utilizando a arquitetura MVC (Model-View-Controller).

## Funcionalidades

O sistema possui as seguintes funcionalidades principais:

- **Autenticação**: Sistema de login para acesso restrito.
- **Gestão de Usuários**: Cadastro e administração de usuários do sistema.
- **Gestão de Produtos**: Cadastro, edição e visualização de produtos disponíveis no petshop.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- **app/**: Contém a lógica da aplicação (Controllers, Models, Views).
- **public/**: Arquivos públicos e ponto de entrada da aplicação.
- **vendor/**: Dependências gerenciadas pelo Composer.
